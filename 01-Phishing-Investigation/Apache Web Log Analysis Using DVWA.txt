7. Log Analysis

Take one log entry like yours:

127.0.0.1 - - [05/Jul/2026:08:58:42 +0530] "GET /DVWA/vulnerabilities/javascript/ HTTP/1.1" 200 3694 "http://localhost/DVWA/vulnerabilities/csp/" "Mozilla/5.0..."

Then explain it in a table.

Field	Meaning
Source IP	127.0.0.1
Method	GET
Requested URL	/DVWA/vulnerabilities/javascript/
Status Code	200
Bytes	3694
Referrer	CSP page
Browser	Firefox 130

8. Findings

For example:

Apache successfully recorded every HTTP request.
The source IP was localhost (127.0.0.1).
Most requests returned HTTP 200.
Firefox was used to access DVWA.
Referrer headers showed navigation between DVWA modules.
